--- Addition 1: DRILL-04 depth-1 guard in validate_manifest_shape() ---

In validate_manifest_shape(), after the existing block that checks EXPECTED_MANIFESTS
existence (the "missing = ..." line) and BEFORE the manifests loop that iterates
over MANIFEST_DIR.glob("*.yaml"), add the following guard:

    # DRILL-04: drill manifests must live in a subdirectory, never at depth-1.
    # CD glob: find k8s/staging -maxdepth 1 -name '*.yaml' — would accidentally
    # schedule the drill Job on every deploy if any drill yaml leaks here.
    drill_depth1 = [
        f for f in MANIFEST_DIR.glob("*.yaml")
        if any(tok in f.stem.lower() for tok in ("drill", "restore-drill"))
    ]
    require(
        not drill_depth1,
        "DRILL-04 violation: drill manifests must be in a subdirectory "
        f"(k8s/staging/restore-drill/), not depth-1; found: "
        + ", ".join(f.name for f in drill_depth1),
    )

NOTE on placement precision: insert this block AFTER the line
  `missing = [name for name in EXPECTED_MANIFESTS if not (MANIFEST_DIR / name).is_file()]`
  `require(not missing, ...)`
and BEFORE the line
  `manifests: list[tuple[str, str, str]] = []`

--- Addition 2: restore-drill.sh bash -n check in validate_scripts() ---

In validate_scripts(), after the existing loop that checks "scripts/backup-postgres-now.sh",
add a second entry to the scripts list so both scripts are checked:

Change the line:
  for script in ["scripts/backup-postgres-now.sh"]:

To:
  for script in ["scripts/backup-postgres-now.sh", "scripts/restore-drill.sh"]:

This means if scripts/restore-drill.sh does not yet exist (e.g. Plan 08-01 has not run),
validate_scripts() will raise an OSError / fail the bash -n check. That is acceptable
